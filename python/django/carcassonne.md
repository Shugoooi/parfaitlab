## 処理の流れ
### 大きな流れ
1. 各自ログイン
2. websocketで相互通信
3. 人が集まったらゲーム開始
4. ゲーム中…
5. 解散

### ゲームの流れ
1. 順番を決める
2. ターン開始
3. カードを引く
4. カードを置く
5. 土地判定
6. 終了判定
7. 2-6を繰り返す
